# JeeraTicketManagement
A clone to manage your activity in prioritywise orders
In this project you can save your daily task works based on yours priority.
Also in this we are using local storage of browser so that we can store users data.

//Facilties
//TO DELETE
also once you complete your task you can delete it with just a double tap on your task sheet.

//TO ADD
just click the plus icon to add new sheet to your application.

//To check priority work
just click on priority color bar on top of page to select priority task you want to do.
